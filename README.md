# 🌍 Carbon Emission Prediction

## 📌 Project Overview
This project predicts carbon emissions based on energy consumption data using machine learning techniques.

## 📊 Dataset
- File: `Country_Energy_Data.csv`
- Contains:
  - Country
  - Year
  - Coal
  - Oil
  - Gas
  - Nuclear
  - Renewables
  - Total Energy

## ⚙️ Features
- Data preprocessing
- Energy consumption analysis
- Carbon emission prediction
- Visualization using graphs

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Streamlit
- Machine Learning

### 1. Install dependencies
